How it works
01
Wrap your agent in one line
Add the @credit decorator to your existing agent function. Float registers it automatically on first run.
02
Agent draws credit when it needs to pay
Float intercepts outgoing payments via x402. The agent draws from its credit line, the task continues. Nothing else changes.
03
Score builds. Limit grows. You watch.
After every repayment cycle, the score recalculates. Cross a threshold and the credit line upgrades automatically. You get a notification.
Drop-in integration
# before Float
async def run(task):
    return await agent.run(task)

# with Float
from Float_sdk import credit

@credit("research-agent-01")
async def run(task):
    return await agent.run(task)
